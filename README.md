# Галактический вестник

Мини-сайт новостей на PHP 8.3 и MySQL 8. Реализованы список с пагинацией по четыре записи, постоянный баннер последней новости и страница деталей новости. Код организован как небольшое MVC-приложение без сторонних фреймворков.

## Работающий сайт

- [Ссылка на сайт](https://galactic-news.example.com/)

## Структура

- `public/index.php` — единая публичная точка входа;
- `public/assets`, `public/images` — публичные стили, шрифты и изображения;
- `src/Controller` — обработка списка и детальной страницы;
- `src/Model` — сущность, контракт репозитория и запросы к MySQL через PDO;
- `src/Core` — подключение к БД, валидация HTTP-входа и рендеринг;
- `views` — PHP-шаблоны;
- `tests/run.php` — автономные проверки.
